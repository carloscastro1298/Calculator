# CALCULATOR
Calculator in Python 
By: Carlos Alberto Castro Martinez
Clan: 8 - Cohorte 5

## DESCRIPTION
This project is a simple calculator created with Python.
The program asks the user to enter two numbers and then choose a mathematical operation.
The calculator can perform:
Addition
Subtraction
Multiplication
Division


## HOW IT WORKS
The program asks the user to enter the first number.
Then it asks for the second number.
The program shows a menu with the available operations.
The user selects the operation.
The program calculates and prints the result.
Operations available
Addition
Subtraction
Multiplication
Division
Technologies used
Python
Basic input/output
Conditional statements (if, elif, else)


## EXAMPLE
Input:

Enter first number: 5
Enter second number: 3

Select operation:
1. Addition
2. Subtraction
3. Multiplication
4. Division

Output:

## Result: 8
